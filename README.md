# Bosta Shipment Track

## Installation

1. Clone the repository:

    ```
    git clone https://github.com/your-username/fast-order-pizza.git
    ```

2. Navigate to the project directory:

    ```
    cd fast-order-pizza
    ```

3. Install dependencies:

    ```
    npm install
    ```

4. Start the development server:

    ```
    npm run dev
    ```
